À propos de Base64

Base64 est un système d'encodage qui convertit des données binaires en texte ASCII. Il utilise 64 caractères différents (A-Z, a-z, 0-9, +, /) pour représenter les données.

Utilisations courantes :

  • Transmission de données binaires par email
  • Encodage d'images dans HTML/CSS (Data URIs)
  • Stockage de données dans JSON ou XML
  • Authentification HTTP Basic

Note : Base64 n'est PAS une méthode de chiffrement. Les données encodées peuvent être facilement décodées. N'utilisez pas Base64 pour protéger des données sensibles.
